
# OrangeHRM Selenium Automation Project

This is a **basic login automation project** that automates logging into the OrangeHRM demo site using Selenium WebDriver with Java and Maven.

# How to Run

### 1. Install required software

- **Java JDK 8+** (check by running `java -version` in terminal)  
  Download: https://www.oracle.com/java/technologies/javase-downloads.html

- **Eclipse IDE for Java Developers**  
  Download: https://www.eclipse.org/downloads/

- **Google Chrome browser** (latest version)  
  Download: https://www.google.com/chrome/

- **ChromeDriver** matching your Chrome version  
  Download: https://chromedriver.chromium.org/downloads  
  Put `chromedriver.exe` somewhere and add its folder to your system PATH or note the path.
